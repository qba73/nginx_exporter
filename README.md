# nginx_exporter

NGINX (OSS) Exporter for Prometheus

## Development

```bash
$ make

Usage:
  help                      Show help message
  dox                       Run tests with gotestdox
  test                      Run tests
  vet                       Run go vet
  check                     Run staticcheck analyzer
  cover                     Run unit tests and generate test coverage report
  tidy                      Run go mod tidy
```
